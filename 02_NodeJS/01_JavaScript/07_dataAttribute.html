Data attribute

자바스크립트 변수에 저장하지 않고 HTML과 관련된 데이터를 저장하는 공식적인 방법으로 HTML 태그의 속성으로, data- 로 시작하는 것들을 넣어준다. 화면에 나타나지는 않지만 웹앱 구동에 필요한 데이터들이며, 이 데이터들을 사용해 서버에 요청을 보내게 된다. 자바스크립트로 쉽게 접근 할 수 있다는 장점이 있다. 반면 dataset에 데이터를 넣어도 HTML 태그에 반영된다. dataset.monthSalary = 10000; 을 넣으면 data-month-salary="10000" 라는 속성이 생긴다.
