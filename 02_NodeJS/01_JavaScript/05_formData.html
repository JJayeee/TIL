FormData

HTML form 태그의 데이터를 동적으로 제어할 수 있는 기능. 주로 AJAX와 함께 사용 된 다. formData.append('name', 'zerocho') formData.append('item', 'orange') formData.append('item', 'melon') formData.has('item') // true formData.has('money') // false fromData.get('item') // orange formData.getAll('item') // ['orange', 'melon'] formData.append('test', ['hi', 'zero']) formData.get('test') // hi, zero formData.delete('test') formData.get('test') // null formData.get('item', 'apple') formData.getAll('item') // ['apple']
